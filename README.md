# Bingo
Uma aplicação simples que tem por objetivo principal simular sorteios de bingo.
## Descrição
Esse app sorteia números de bingo aleatóriamente, sem repetir, e mostra o número sorteado(em conjunto com a letra da casa correspondente)
![print1](/data/prints/print1.jpg)
e mostra uma tabela ao lado com os números já sorteados(para agradar os desatentos)
![print1](/data/prints/print2.jpg)
a cada sorteio são rufados os tambores antes de aparecer o número, ao apertar o botão o jogo reseta.
---
## Objetivo
Minha familia queria jogar bingo hoje, então pensei: "se querem bingo, o bingo eu darei" e assim fiz esse aplicativo, testando a minha velocidade de desenvolvimento ao mesmo tempo em que entrego algo a miha familia. O app foi desenvolvido para que todos pudessem participar e se divertir.
